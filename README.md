A personal portfolio website showcasing my work as a frontend developer with a UX-focused approach to building accessible, thoughtful interfaces.

## 🌐 Live Site

[View Portfolio](https://tara-olivia.netlify.app/)

## 📋 Description

This portfolio website features my projects, skills, work experience, and certifications. Built with a focus on clean design, accessibility, and performance.

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6 modules)
- Deployed on Netlify

## 🎨 Features

- Responsive design for all devices
- Dynamic image carousel with 200+ personal photos
- Interactive navigation with smooth scrolling
- Tabbed experience section
- Project showcase with live demos and GitHub links
- Glassmorphism design aesthetic

## 🚀 Getting Started

### Prerequisites

- A modern web browser
- (Optional) A local server for development

### Installation

1. Clone the repository
```bash
git clone https://github.com/taraolivia/portfolio.git
```

2. Navigate to the project directory
```bash
cd portfolio
```

3. Open `index.html` in your browser or use a local server
