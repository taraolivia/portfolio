Project Overview

Bloom is a responsive blog platform targeting users interested in lifestyle and self-care, offering a modern and clean interface. Developed using HTML, CSS, and JavaScript, the site integrates an API to manage dynamic blog posts. Key features include a carousel for the 3 latest posts, a responsive grid displaying all posts, and admin functionality for registration, login, and post management.

Key Technologies

  • HTML, CSS, JavaScript
  • API Integration
  • Responsive Grid Layout
  • User Authentication
  • Admin Dashboard
